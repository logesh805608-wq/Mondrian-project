# Piet Mondrian Art Generator

A web development project recreating the famous abstract painting *Composition with Red, Blue, and Yellow* by Piet Mondrian. This project was built to master absolute layouts, track dimensions, and structural properties using CSS Grid.

## 🛠️ Tech Stack
* **HTML5:** Structural element framing.
* **CSS3:** Advanced CSS Grid, track spanning (`grid-area`, `span`), and modern CSS Box Model interactions.

## 📐 Core Concepts Learned
* **CSS Grid Tracks:** Carving out a precise canvas using custom row and column measurements.
* **Grid Area Positioning:** Mapping layout elements across specific horizontal and vertical line coordinates.
* **Box Model Constraints:** Visualizing how element gaps and overlapping borders influence overall layout spacing.
